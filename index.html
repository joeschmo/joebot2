joebot2

A fully customizable Haskell IRC bot. Running on irc.freenode.net #roboclub.

Written in Haskell

joebot uses the latest in Hackage Library support, including Control.Lens. The Core of joebot is designed to be small so that it's easier to modify the inner workings of joebot.

Open Source

joebot's source is given under the MIT license, allowing anyone to fork, modify, and use joebot as they please.

Modular Design

The plugin system was written to make writing addons as painless as possible. Many of joebot's commands are written as plugins, keeping the Core clutter-free.
