joebot2

A fully customizable Haskell IRC bot. Running on irc.freenode.net #roboclub.

Written in Haskell

joebot was written in Haskell, allowing for rapid development while enjoying great robustness. joebot uses the latest Hackage libraries, including Control.Lens. The Core of joebot is designed to be small so it's easier to modify the inner workings of joebot.

Open Source

joebot's source is covered by the MIT license, allowing anyone to fork, modify, and use joebot as they please.

Modular Design

The plugin system was written to make writing addons as painless as possible. Many of joebot's commands are written as plugins, keeping the Core clutter-free. Documentation on how to write new plugins is provided here.
