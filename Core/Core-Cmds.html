write :: Text -> Text -> Net ()

Generic write to socket

privmsg :: Text -> Maybe Text -> Text -> Net ()

Used to send PRIVMSG responses to the socket. 

 privmsg nick channel message.

If channel is Nothing, then it will send the message as a private message to nick.

action :: Text -> Net ()

Used to send ACTION responses to the socket.
